# 活动创建功能更新

## 更新日期
2026-03-11

## 更新内容

### 1. 移除结束时间字段
- 原来：活动结束时间（用于计算活动时长）
- 现在：只保留活动日期和开始/结束时分

### 2. 新增报名截止时间
- 字段名：`registration_deadline`
- 类型：日期选择器
- 作用：设置报名截止的日期，时间到了后自动截止报名
- 验证：必填项

### 3. 新增跑步类型
- 字段名：`run_type`
- 类型：单选按钮组
- 选项：
  - `road` - 路跑
  - `trail` - 越野跑
  - `hiking` - 徒步
  - `brand` - 品牌合作跑
- 验证：必填项
- 默认值：`road`（路跑）

### 4. 新增 Dress Code
- 字段名：`dress_code`
- 类型：文本输入框
- 说明：例如统一穿跑团队服、主题装扮等
- 验证：选填项
- 最大长度：50 字符

## 数据结构变更

### 活动表 (activities) 新增字段

```javascript
{
  // ... 原有字段
  run_type: String,              // 跑步类型：road | trail | hiking | brand
  dress_code: String,            // Dress Code 要求
  registration_deadline: String  // 报名截止日期：YYYY-MM-DD
}
```

## 页面变更

### 创建活动页面 (activity-create)
- 移除结束时间日期选择器
- 新增报名截止日期选择器
- 新增跑步类型单选按钮组
- 新增 Dress Code 文本输入框
- 优化时间选择器布局（开始/结束时间并排显示）

### 活动详情页面 (activity-detail)
- 显示跑步类型（中文文本）
- 显示 Dress Code（如果有）
- 显示报名截止时间

## 使用示例

### 创建路跑活动
```
标题：周末晨跑
跑步类型：路跑
Dress Code: 统一穿跑团队服
活动日期：2024-03-17
开始时间：08:00
结束时间：10:00
报名截止：2024-03-16
名额：30 人
积分：20
```

### 创建越野跑活动
```
标题：香山越野跑挑战
跑步类型：越野跑
Dress Code: 建议穿越野跑鞋和速干衣
活动日期：2024-03-24
开始时间：07:00
结束时间：10:00
报名截止：2024-03-23
名额：20 人
积分：35
```

## 后续优化建议

1. **自动截止报名**：在云函数中定时检查 `registration_deadline`，时间到达后自动将活动状态设为 `registration_closed`

2. **报名截止提醒**：活动截止前 2 小时发送订阅消息提醒未报名的团员

3. **跑步类型图标**：为每种跑步类型设计专属图标，提升视觉识别度

4. **Dress Code 模板**：提供常用 Dress Code 模板快速选择
