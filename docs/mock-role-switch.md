# Mock 角色切换指南

## 快速切换角色

### 方法一：修改配置文件（推荐）

1. 打开 `miniprogram/app.js`
2. 修改 `CURRENT_ROLE` 值：
   ```javascript
   App({
     USE_MOCK: true,
     CURRENT_ROLE: 'leader',  // 'member' | 'leader' | 'admin'
     // ...
   });
   ```
3. 保存后重新编译小程序

### 方法二：使用调试页面

1. 进入「我的」页面
2. 点击底部「角色切换（Mock）」菜单
3. 选择要切换的角色：
   - **团员**：普通团员权限
   - **团长**：团长权限（最高权限）
   - **管理员**：管理员权限

## 角色说明

### 团员 (member)
- 查看活动列表
- 报名参加活动
- 查看积分
- 积分兑换商品
- 查看订单

### 管理员 (admin)
- 所有团员权限
- 发布活动
- 审核新成员
- 活动签到确认
- 管理商品
- 管理订单

### 团长 (leader)
- 所有管理员权限
- 任命/撤销管理员
- 解散跑团
- 设置规则
- 查看所有统计数据

## 切换角色后的操作

切换角色后，建议：
1. 退出当前小程序
2. 重新进入小程序
3. 或点击调试页面的「清除缓存并重启」按钮

## 注意事项

1. Mock 模式仅在 `USE_MOCK = true` 时有效
2. 切换角色会清除当前登录状态
3. Mock 数据不会持久化，重启后恢复初始状态
4. 正式环境需要切换到云开发模式（`USE_MOCK = false`）
