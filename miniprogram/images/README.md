## 图标来源建议

1. [阿里巴巴矢量图标库](https://www.iconfont.cn/) - 丰富的图标资源
2. [Flaticon](https://www.flaticon.com/) - 高质量图标
3. 自行设计 - 保持风格统一

## 注意事项

- 所有图片使用 PNG 格式
- 图标建议使用 SVG 或 PNG 格式
- 保持图标风格统一
- 建议提供 @2x 和 @3x 分辨率版本
